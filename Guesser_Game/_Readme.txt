Guesser Game

Rules => 
1. First of all, Participant have to select Minimum and Maximum values to generate random number between Minimum and Maximum values (Default Minimum values is Zero).
2. Also diffrence between Minimum and Maximum number should greater than 5.
3. After you need to Guess the random number, which is generated in range of between Minimum and Maximum values.
4. Participant will have only 5 chances to Guess the Number.
5. If Participant selected "Get Help!", then he/she will have only one chance among 2 options in order to find correct number(if only chances are greater than 3).
