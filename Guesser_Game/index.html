Rules for Guessing Game

  • First of all, Participant have to select Minimum and Maximum values to generate random number between Minimum and Maximum values (Default Minimum values is Zero).
  • Also diffrence between Minimum and Maximum number should greater than 5.
  • After you need to Guess the random number, which is generated in range of between Minimum and Maximum values.
  • Participant will have only 5 chances to Guess the Number.
  • If Participant selected "Get Help!", then he/she will have only one chance among 2 options in order to find correct number(if only chances are greater than 3).
